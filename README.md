# Custom_Arcade_Pcb
Arduino based custom pcb design for arcade machine
The custom PCB layout is Arduino nano based/ Changes can be made for using Nano esp in future project 
The basic design integrates the following subsystems into a unified board:
The board includes a Timer pinout, a Sensor pin, StartB pin, 2 Relay outputs, an LEDs pinout, a ticket pinout, extra pinouts, and power pins
Software used: Kicad 9.0
